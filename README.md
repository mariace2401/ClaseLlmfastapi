CHAT BOT PSICOLOGICO
REQUERIMIENTOS FUNCIONALES
-El sistema debe permitir que el usuario cree una cuenta o incia sesion de forma segura
-El chat bot deb tener conversaciones profundas cn el usuario mediante lo que el usuario escriba
-El sistema debe identificar emociones basicas a partir del texto ingresado
-El chatbot puede sugerir ejercicios como respiracion, tecnicas de relajacion o un diario dependiento la situacion del usuario
-Si el sistema detecta palabras de autoloseiones o de crisis, el chat ayudara con un mensaje de emergencia y mostrando las lineas de ayuda profesional
REQUERIMIENTOS NO FUNCIONALES
-Los datos del usuario debe almacenarse de forma segura y cumplir con la proteccion de los datos
-El sistema debe ser 24/7, estar disponible en todo momento sin interrupciones para el usuario
-El chatbot en cada respuesta debe tardar menos de 3 segundos en responder
-La interfaz del sistema debe ser sencilla, intuitiva y facil de usar para personas mayores de 14 años
-El sistema debe soportar muchos usuarios conectados al mismo tiempo sin disminuir el rendimiento
